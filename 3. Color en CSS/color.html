Propiedades de Color en CSS

El color de texto puede cambiarse con la propiedad color. Este texto es de color rojo.

Este texto es de color verde.

El color de fondo de un elemento se cambia con la propiedad background-color. Este párrafo tiene fondo azul.

Este párrafo tiene fondo amarillo.

La opacidad se puede aplicar usando valores RGBA. Este párrafo tiene un fondo verde con 50% de opacidad.

Los colores se pueden representar en formato hexadecimal (#RRGGBB). Este párrafo tiene un fondo de color naranja (#FF5733).

Los colores también se pueden definir en formato HSL (Hue, Saturation, Lightness). Este párrafo tiene un fondo de color verde brillante en formato HSL.

El formato RGB (Red, Green, Blue) es otra forma de definir colores. Este párrafo tiene un fondo de color rojo en formato RGB.
