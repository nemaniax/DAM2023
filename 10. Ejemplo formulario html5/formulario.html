Formulario de Contacto

Estas características hacen que la creación y validación de formularios sea más fácil y accesible en HTML5.

Comentarios:

  • El elemento <form>: HTML5 introduce el elemento <form> que se utiliza para crear formularios web. Es un contenedor para campos de entrada y otros elementos relacionados con formularios.
  • Atributo "required": En cada campo de entrada (nombre, correo electrónico y mensaje), se utiliza el atributo "required". Esto indica que esos campos son obligatorios y no se puede enviar el formulario si están vacíos.
  • Tipo de entrada "email": El campo de correo electrónico utiliza el tipo de entrada "email", que ayuda a los navegadores a validar automáticamente si el contenido es una dirección de correo electrónico válida.
  • Etiquetas (<label>): Se utilizan etiquetas <label> para describir los campos de entrada. Esto mejora la accesibilidad y la usabilidad del formulario, ya que los usuarios pueden hacer clic en la etiqueta para enfocar el campo correspondiente.
  • Área de texto (<textarea>): Se utiliza el elemento <textarea> para el campo "Mensaje". El atributo "rows" se utiliza para especificar el número de filas visibles en el área de texto.
  • Botón de envío (<input type="submit">): El botón de envío permite al usuario enviar el formulario. El texto del botón es "Enviar". Cuando se hace clic en este botón, el navegador intentará enviar el formulario.

En resumen, estas son algunas de las novedades de HTML5 en este formulario:

  • El uso de etiquetas <label> para describir los campos.
  • El atributo "required" para hacer campos obligatorios.
  • El tipo de entrada "email" para la validación automática del correo electrónico.
  • El uso de <textarea> para campos de texto más largos.

Estas características hacen que la creación y validación de formularios sea más fácil y accesible en HTML5.
