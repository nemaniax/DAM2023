Ejemplo de Medidas en CSS

En CSS, puedes utilizar diferentes unidades de medida para especificar tamaños. Algunas de las unidades más comunes son:

  • Píxeles (px): 20px
  • Porcentaje (%): 50%
  • Viewport Width (vw): 30vw
  • Viewport Height (vh): 40vh
  • Em (em): 2em
  • Rem (rem): 1.5rem
